Mengubah Aplikasi SIASATI On 5 Jun 2019

Aplikasi SIASATI di http://siasati.dephub.go.id sepertinya harus diubah, secara fungsional masih dapat berjalan dengan baik untuk database masih dengan menggunakan MySQL tidak ada masalah dengan hal itu tetapi untuk penyajian data grafik-grafik akan lebih baik menggunakan database yang menyediakan fungsi itu seperti PostgreSQL. Rencananya semua aplikasi akan dipisah-pisah seperti API, Frontend dan Backend. API untuk penyedia data untuk Frontend dengan menggunakan teknologi GO Rest sedangkan Frontend akan menyajikan data dashboard menggunakan teknologi Angular 6 dan Backend akan menggunakan Laravel 5.8
